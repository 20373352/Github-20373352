# 软件工程基础个人仓库 - 百度首页模仿

## 🚀 项目概述

本项目是**北京航空航天大学软件工程基础课程**的个人实验仓库。核心内容是一个基于纯前端技术栈（HTML5, CSS3, JavaScript (Vanilla)）实现的**高仿百度搜索首页**。

该项目旨在通过模仿真实世界的应用界面，实践前端开发基础技能，包括页面布局、样式设计和基础交互逻辑。

## ✨ 核心功能亮点

*   高仿百度搜索首页界面
*   自定义热榜列表展示（游戏主题）
*   模拟用户登录/登出功能（通过 Kernal 模块）
*   搜索按钮点击事件监听

## 🛠️ 技术栈

| 类别 | 技术名称 | 备注 |
| :--- | :--- | :--- |
| **标记语言** | HTML5 | 负责页面结构 |
| **样式设计** | CSS3 | 负责页面美化和布局 |
| **交互逻辑** | JavaScript (Vanilla) | 负责用户事件处理和模拟登录状态管理 |

## 📂 文件结构概览

本项目的主要代码位于 `软工第二次上机前端/` 目录下：

| 文件路径 | 描述 |
| :--- | :--- |
| `软工第二次上机前端/index.html` | 主页面结构，包含热榜列表和搜索框。 |
| `软工第二次上机前端/附件/index.js` | 页面交互逻辑，包含登录/登出和搜索事件处理。 |
| `软工第二次上机前端/附件/style.css` | 页面样式，实现百度首页的布局和视觉效果。 |
